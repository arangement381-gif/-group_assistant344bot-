# 🤖 Group Assistant 344 Bot

Telegram bot for group management and automation.

## 🚀 Features

- 📊 Group statistics and information
- 👥 Member management
- 🔧 Admin commands
- 💬 Auto-replies to common messages
- 📈 Database storage
- 🎨 User-friendly interface

## 🛠️ Tech Stack

- **Language:** Python 3.11
- **Framework:** python-telegram-bot v20+
- **Database:** SQLite (PostgreSQL ready)
- **Deployment:** Railway
- **Version Control:** GitHub
